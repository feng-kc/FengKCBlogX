# 欢迎来到 BlogX

这是一个现代化的个人博客平台，支持Markdown格式的文章展示。

## 功能特性

- ✅ 完美的Markdown渲染支持
- ✅ 代码语法高亮和复制功能
- ✅ 响应式设计，支持移动端
- ✅ 明暗主题切换
- ✅ 文章搜索功能
- ✅ 表格和复杂元素支持

## Python示例

```python
def fibonacci(n):
    """生成斐波那契数列"""
    a, b = 0, 1
    for _ in range(n):
        yield a
        a, b = b, a + b

# 使用示例
for num in fibonacci(10):
    print(num)
```
